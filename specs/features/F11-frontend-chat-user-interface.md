# F11: Frontend Chat User Interface

## Goal
Develop a responsive React application that provides a user-friendly chat interface to interact with the backend chatbot API.

## Acceptance Criteria
- [ ] A main chat page is rendered, displaying a message input field and a history of chat messages.
- [ ] Users can type a message, submit it, and see it appear in the chat history.
- [ ] The frontend sends user questions to the POST /chat endpoint (F03) and displays the chatbot's response in the chat history.
- [ ] The user interface is responsive across common screen sizes.

- frontend/src/components/ChatWindow.js
- frontend/src/components/MessageInput.js
