# F07: React User Chat Interface

## Goal
Develop a user-friendly React frontend application that allows users to interact with the medical FAQ chatbot by submitting questions and displaying the corresponding answers.

## Acceptance Criteria
- [ ] The chat interface provides an input field for users to type questions.
- [ ] User-submitted questions are sent to the chatbot API (F04).
- [ ] Chatbot answers are displayed clearly in the interface.
- [ ] The interface maintains a history of the current conversation (questions and answers).
- [ ] The UI is responsive and provides a good user experience.

- frontend/src/pages/Chatbot.js
- frontend/src/components/ChatWindow.js
