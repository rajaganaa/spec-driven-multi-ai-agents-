# F02: Backend Foundation & Security Defaults

## Goal
Set up the core FastAPI application, PostgreSQL database connection, SQLAlchemy models, and configure essential security practices for the backend.

## Acceptance Criteria
- [ ] A FastAPI application is initialized and runs successfully.
- [ ] A PostgreSQL database connection is established and validated.
- [ ] SQLAlchemy models for users and FAQs are defined.
- [ ] CORS middleware is configured to allow frontend access.
- [ ] Environment variables are used for sensitive configurations (e.g., database credentials, secret keys).
