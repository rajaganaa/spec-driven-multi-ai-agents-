# F06: React Admin Dashboard

## Goal
Build a React frontend application for authenticated administrators to securely log in, view, create, update, and delete medical FAQ entries.

## Acceptance Criteria
- [ ] The admin dashboard provides a login page for administrators.
- [ ] Authenticated administrators can view a list of all FAQs.
- [ ] Administrators can add new FAQ entries via a form.
- [ ] Administrators can edit existing FAQ entries.
- [ ] Administrators can delete FAQ entries.
- [ ] The dashboard handles authentication tokens for secure API calls.
