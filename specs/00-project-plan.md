# Project Plan

## Goal
Build a HIPAA-aware medical FAQ chatbot using a Python/FastAPI backend and a React frontend.

## Summary
The project will create a stateless web application featuring a React frontend and a FastAPI backend. The core of the application is a simple, non-AI chatbot that matches user queries to a predefined set of medical FAQs. To ensure HIPAA awareness, the system will be designed to be completely stateless, will not log or store any user queries or personal data, and the user interface will prominently feature disclaimers against entering personal health information.
