# Project Plan

## Goal
Build a HIPAA-aware medical FAQ chatbot using FastAPI (Python), React, and PostgreSQL, ensuring proper handling of sensitive information.

## Summary
This project will develop a HIPAA-aware medical FAQ chatbot. The backend, built with FastAPI and PostgreSQL, will manage FAQ content and provide a chatbot interaction API. The frontend, developed with React, will offer a user interface for both chatbot interaction and browsing FAQs, with explicit safeguards to prevent the input and storage of Personal Health Information (PHI).

## Features
- F14: Backend Core & Database Setup — assigned to backend
- F15: FAQ Content Management API — assigned to backend
- F16: Chatbot Interaction API (HIPAA-Aware) — assigned to backend
- F17: React Frontend Foundation — assigned to frontend
- F18: Chatbot User Interface & FAQ Display — assigned to frontend
